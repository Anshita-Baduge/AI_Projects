# Smart Study Planner using Artificial Intelligence

This project demonstrates the application of Artificial Intelligence concepts

and constraint satisfaction.

## AI Concepts Used
- UNIT-I: Intelligent Agent and Problem Formulation
- UNIT-II: Uninformed and Heuristic Search
- UNIT-III: Constraint Satisfaction
